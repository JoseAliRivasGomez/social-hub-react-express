# Social Hub Backend

1. Clonar proyecto
2. __npm i__
3. Clonar el archivo __.env.template__ y renombrarlo a __.env__
4. Cambiar las variables de entorno
5. Levantar la base de datos
```
docker-compose up -d
```
6. Levantar: __npm run dev__
7. Ejecutar Seed
```
http://localhost:4000/api/seed
```
